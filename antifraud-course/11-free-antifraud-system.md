# 11. Топовая антифрод-система бесплатно

**Автор:** Вектор Т13
**Длительность:** ~8 минут
**Уровень:** Практический
**Тип урока:** Хак для доступа к Enterprise антифроду

---

## Краткое содержание

Показан способ получить бесплатный доступ к IP Quality Score Enterprise (тариф от $5000/месяц) через уязвимость на сайте Indeed.com. Полноценная проверка IP, Device ID, Fraud Chance без регистрации.

---

## Ключевые тезисы

### 1. IP Quality Score тарифы

| Тариф | Цена | Особенности |
|-------|------|-------------|
| Free | $0 | Базовая проверка |
| Startup | $49/мес | Только количество запросов |
| SMB Basic | $249/мес | Premium Blacklists |
| SMB Plus | $499/мес | Residential Proxy Detection |
| **Enterprise** | **$5000+/мес** | **Fraud Fusion, Abuse Shield, Dark Data, Botnets** |

### 2. Что даёт Enterprise

- **Fraud Fusion** - история IP-адреса
- **Abuse Shield** - защита от злоупотреблений
- **Dark Data** - данные с прокси-сервисов
- **Emerging Botnets** - обнаружение новых ботнетов (Luxox, 911 и др.)

### 3. Способ получения

Уязвимость на сайте **Indeed.com** - прямой доступ к IP Quality Score CDN.

## Полный конспект

### Почему не публичные чекеры

> "Все публичные чекеры являются ханипотами. Я множество раз уже говорил, не буду повторять."

### IP Quality Score - что это

- Антифрод-система для проверки IP
- Проверка кредитных карт
- Проверка номеров телефонов
- Партнёр VectorT13 много лет

### Разница между тарифами

#### Free (бесплатный)
- Базовая проверка IP
- Тысячи запросов в месяц
- **Чистый IP может показать грязный на платном тарифе**

#### SMB Basic ($249)
- Premium Blacklists
- Расширенные блэк-листы
- Дополнительные источники информации

#### SMB Plus ($499)
- Residential Proxy Detection
- Обнаружение резидентных прокси

#### Enterprise ($5000+)
- **Fraud Fusion** - полная история IP
- **Abuse Shield** - защита от абуза
- **Dark Data** - данные с даркнета
- **Emerging Botnets** - новые ботнеты
- Реагирует на Luxox, 911 и аналоги

### Метод получения доступа

#### Шаг 1: Перейти на Indeed.com

Indeed - сайт поиска работы, использует IP Quality Score Enterprise.

#### Шаг 2: Открыть DevTools

```
F12 -> Network -> All
```

#### Шаг 3: Перейти на Sign In

Очистить лог, перейти на страницу Sign In.

#### Шаг 4: Найти запрос

Фильтр: Third-party resources
Искать: Fetch запрос к IP Quality Score CDN

#### Шаг 5: Открыть URL

Двойной клик на запросе = прямой доступ к Enterprise API.

### Что показывает

| Параметр | Описание |
|----------|----------|
| **Fraud Chance** | Вероятность фрода (главный параметр) |
| **Device ID** | Идентификатор устройства |
| Proxy/VPN/TOR | Обнаружение прокси |
| Country | Страна IP |
| SSL Fingerprint | Отпечаток SSL |
| Hardware Check | Проверка железа |

### Калибровка антидетекта

Device ID показывает:
- Новое устройство = "unknown"
- Известное устройство = ID

**Использование:** Проверить, видит ли система вас как нового пользователя.

### Наблюдение пользователя

> "Если использовать macOS, fraud chance падает."

### Google Dorks

Этот метод не уникален для Indeed:
- Можно найти другие сайты с такой же уязвимостью
- Google Dorks из урока по анализу антифрод-систем
- Возможно найти даже ThreatMetrix

## Технические детали

### URL структура

```
https://[ipqs-cdn].ipqualityscore.com/api/...?key=...&device=...
```

### Network запрос

```
Type: Fetch
Domain: *.ipqualityscore.com
Response: JSON с полной информацией
```
